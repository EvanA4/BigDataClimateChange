# BigDataClimateChange
A Python-based approach to answer the following question: how have countries' daily temperature ranges changed over time?

### How to Use
**You must first download the necessary data!** NaturalEarthData contains [download links](https://www.naturalearthdata.com/downloads/110m-cultural-vectors/) to the necessary country data. Just download [this](https://naciscdn.org/naturalearth/110m/cultural/ne_110m_admin_0_countries.zip) country data ZIP file specifically. Extracting the ZIP file should give you the following files:

```
ne_110m_admin_0_countries.cpg
ne_110m_admin_0_countries.prj
ne_110m_admin_0_countries.shp
ne_110m_admin_0_countries.VERSION.txt
ne_110m_admin_0_countries.dbf
ne_110m_admin_0_countries.README.html
ne_110m_admin_0_countries.shx
```

Place `ne_110m_admin_0_countries.shp` into the root directory of your project.
