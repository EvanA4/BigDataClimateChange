# BigDataClimateChange
A Python-based approach to answer the following question: how have countries' daily temperature ranges changed over time?

### How To Run
Download and extract all files from [this shapefile download](https://www.arcgis.com/home/item.html?id=fa510018bdd044b08fc64d2a16bc680a) into `/mapdata`. Alternatively, extract all files from [this download](https://www.arcgis.com/home/item.html?id=fa510018bdd044b08fc64d2a16bc680a) for a lower quality shapefile.
